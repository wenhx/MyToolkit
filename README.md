# MyToolkit
在日常使用电脑的过程中，自己写的一些小工具程序。

## AudioFileLength
在Windows系统中统计指定目录下所有音频文件的合计时长。

## FtpUploader
非交互式命令行Ftp上传工具。

##DirectoryEverything  
An [Everything](https://www.voidtools.com/) helper tool uses the SDK of Everything to search all files under the special folder.
一个[Everything](https://www.voidtools.com/) 软件的帮助小工具，它通过Everything的接口来批量查找指定目录下面的所有文件。
